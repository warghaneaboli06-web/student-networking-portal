# Student Professional Networking Portal

## Description
Student Professional Networking Portal is a responsive web application developed using HTML and CSS.  
The website helps students create professional connections, showcase skills, projects, certificates, and explore internship opportunities.

---

## Features

- Responsive Design
- Student Profiles
- Networking System
- Internship & Job Section
- Project Showcase
- Chat System
- Discussion Forum
- Event & Workshop Updates
- Secure Login Interface
- Contact Section

---

## Technologies Used

- HTML5
- CSS3
- Font Awesome Icons
- Visual Studio Code
